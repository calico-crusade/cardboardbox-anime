iOS / MacOS App Install Instructions

Unfortunately, it costs too much to develop an actual iOS app, so instead;

I went with the free Progressive Web App (PWA) alternative.

The installation steps are a little weird, but they're decently easy.

  1. Navigate to this site on the safari app (It HAS to be safari): https://cba.index-0.com
  2. Click the "Share" icon ios_share
  3. Click the "Add to Home Screen" button
  4. Name it whatever you want and click "Add".

Now you should see the app on your home page.

Android App Install Instructions

To install the app on your android, follow these instructions:

  1. Open your favourite android browser (I suggest chrome or firefox)
  2. Navigate to this site: https://cba.index-0.com
  3. Click the settings icon for the page more_vert
  4. Scroll down until you see "Install" or "Install App"
  5. Click the "Install" button

Now you should see the app on your home page.

Windows App Install Instructions

To install the app on your windows device, follow these instructions:

  1. Open Chrome on your computer (It HAS to be chrome): https://cba.index-0.com

  2. Click the install option either in the URL bar or the settings menu (see image below)

    • You should see an icon popup in the URL bar that prompts you to install the application.
    • Alternatively you can click the settings icon and click "Install Cardboard Box..."

Now you should see the app on your desktop.
